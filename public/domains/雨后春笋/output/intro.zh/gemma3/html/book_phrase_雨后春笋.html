Simple rule: 尹 always measures something's deviation from a target or ideal. It's about recognizing where things aren't quite right. Mathematically, we can represent this as:

尹 = |目标值 - 实际值|

Where:

  • 尹 (yūn) is the difference
  • 目标值 (mù biāo zhí) – target value or ideal value
  • 实际值 (shì shí zhí) – actual value

Try it yourself: Let’s say you want to save $100 for a new game (你的目标 - nǐ de mù biāo). You work for a week and manage to save only $65 (你的实际 - nǐ de shì shí). What is the 尹? Calculate: |$100 - $65| = $35. You have a 尹 of $35!
